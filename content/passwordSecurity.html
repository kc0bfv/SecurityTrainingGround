Password Security

Introduction

Many Internet-based services use passwords to protect you online. For example:

  • Facebook uses passwords to prevent someone from impersonating you
  • Dropbox uses passwords to protect data that you want to keep private
  • Amazon uses passwords to prevent other people from buying things using your credit card
  • Banks use passwords to prevent people from spending your money

Other uses for passwords exist, and most services do some combination of these things. Passwords are often the only thing services use to protect you and your data, and therefore they are critical to your security online.

Hackers and Passwords

Passwords have been in the news a lot, recently. Hackers break into companies and steal passwords, hackers guess passwords, hackers trade passwords online... How do hackers do all that?

If a hacker breaks into a company's computer systems, he or she can download the file containing all the passwords. That might look like this:
UserPassword
mikemike123
chris2014chrisIsTheBest!
jimboasdfJimbo
Janejn$5!as334n0

Now the hacker has all the usernames and passwords, so they can easily login as any of those users! That's terrible. Responsible companies store their passwords using a secure method called "hashing". "Hashed" passwords look like gobbeldygook to a hacker, something like this:
UserHashed Password
mikec9d7bf3e1dee4e0a0bd243c86b053b8b
chris20141ecd5271b8aec27669d0abda0c88186c
jimbo43f6589407a536b204111b723de5af1c
Janeb7e478636917a243cc8092824a3e3638

Just let that jerk hacker try to use those! He or she cannot simply login to the company, now, first they have to "crack" the passwords. "Cracking" difficult passwords takes a long time, and involves a computer trying passwords over-and-over-and-over-and-over-an... well, you get the point.

Actually, the hacker will crack mike and jimbo's passwords very easily. Cracking chris2014's password will take much longer, and the hacker will probably never figure out Jane's password. Jane picked a great password, how can you pick passwords like Jane's?

Protecting Yourself

Here are some tips:

Password Do's

  • Choose passwords longer than 8 characters
  • Choose passwords with upper and lowercase letters, numbers, and funky characters (like $, % or #)
  • Consider putting 4 or more words together in a phrase that you will remember
  • Use different passwords for different sites, especially sites with your important data
  • Use a password safe program like 1Password or KeePass to store many difficult passwords, and only have to remember one password

Password Dont's

  • Don't choose passwords that contain your user name, or real name, or birthday
  • Don't choose one or two words as your password ("timeMachine" and "bookLover" are no good)

When hackers crack passwords they tell their computer to guess with all the passwords in the dictionary, then they tell the computer to guess two-word combinations, and combinations with a few numbers thrown in there. These techniques only take a little while to try, even when the list of all the users is very large. Hackers start with those techniques because so many people choose passwords like that. In less than an hour, a hacker would have mike and jimbo's private data, and perhaps they wouldn't go any further than that.

By choosing a password that's fits those "do's" up above, you make it more difficult for a hacker to crack your password. Finally - you should avoid reusing your passwords. If you use a password on one website, don't use it on other websites. This is especially true if you care about the data stored on those websites... If a hacker breaks into one website and gets your password, they can use that password on the other sites. Don't make it easy for hackers!

Unfortunately, it's tough to remember one good password, let alone one for each website. Fortunately, there are several password storage programs out there, like 1Password, KeePass, and LastPass. These are called "Password Safes", or "Password Vaults", or "Password Managers". If you're interested in using one, you should choose one based on the devices it supports, and the devices you use. For example, KeePass has versions that work on Android, iPhone, Windows, Mac, and Linux. Also, it's open-source, so anyone can look at the source code and make sure it keeps your passwords secure instead of emailing them to a nefarious hacker...

Windows Password Settings

There are a few simple things you can do to make Windows more secure by default. The Windows security policies provide a way to require password security for all users. Go into the Control Panel, open Administrative Tools, then Local Security Policy. Some good settings in the Password Policy section are:

  • Enforce Password History: 5
  • Maximum Password Age: 90
  • Minimum Password Length: 8
  • Password Must Meet Complexity Requirements: Enabled
  • Store Password Using Reversible Decryption: Disabled

Additionally, some good settings for the Account Lockout Policy section are:

  • Account Lockout Duration: 5
  • Account Lockout Threshold: 5
  • Reset Account Lockout Counter After: 5
